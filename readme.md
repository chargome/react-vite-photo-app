## description
this is a small showcase on how to setup react with `typescript`, `vite`, `tailwind` and `zustand`

## setup
- `npm i`

## run app
- `npm run dev`

## run tests
- `npm run test`
- the app uses `vitest` for unit tests

## run e2e tests
- `npm run test:e2e`
- the app uses `cypress` for e2e testing
